# Formulario multipaso

## Descripción

Formulario que consta de 3 pasos, cada paso esta gestionado por un state global usando useContext y contiene validaciones.

## Herramientas

```
 - tailwind
 - shadcn
 - react
 - typescript

```

## Comandos

instalación de paquetes

```shell
npm run install
```

ejecutar app

```shell
npm run dev
```
